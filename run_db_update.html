Database Update

To add the children_age_details column to the database, please:

  1. Open your web browser
  2. Navigate to: http://localhost/git/gleesire/update_database.php
  3. This will add the required column to the converted_leads table

Alternatively, you can run this SQL command directly in phpMyAdmin:

ALTER TABLE `converted_leads` ADD COLUMN `children_age_details` VARCHAR(255) NULL AFTER `infants_count`;
